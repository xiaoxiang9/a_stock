# 牧原股份信号定义

## 用途

本文件定义牧原股份第一版研究驱动决策系统的核心信号，并给出每类信号在决策中的作用。

## 信号分层

### A 层：决定大方向的核心信号

| 信号 | 作用 | 对应动作影响 |
| --- | --- | --- |
| 猪价趋势 | 判断行业景气方向 | 决定是否更接近加仓观察或继续持有 |
| 能繁母猪存栏趋势 | 判断未来供给压力 | 决定周期判断是否前移 |
| 月度出栏量与销售均价 | 判断公司经营兑现度 | 验证公司是否跑赢行业 |
| 成本变化 | 判断护城河是否加强或削弱 | 决定长期逻辑是否成立 |

### B 层：辅助判断质地与安全边界的信号

| 信号 | 作用 | 对应动作影响 |
| --- | --- | --- |
| ROE | 判断资本回报质量 | 辅助确认是否仍属好公司 |
| 毛利率 | 判断经营弹性 | 验证盈利恢复质量 |
| 资产负债率 | 判断财务安全边界 | 恶化时提高审慎级别 |
| 经营现金流 / 收入 | 判断利润含金量 | 防止只看账面利润 |

### C 层：交易节奏信号

| 信号 | 作用 | 对应动作影响 |
| --- | --- | --- |
| PE(TTM) / PB | 判断估值位置 | 决定是否接近加仓或减仓观察 |
| 换手率 | 判断拥挤度 | 热度过高时倾向减仓观察 |
| 成交额 / 量比 | 判断短期情绪 | 修正节奏，不改变长期逻辑 |

## 应用顺序

1. 先看 A 层，判断周期和经营是否共振
2. 再看 B 层，确认公司质地有没有受损
3. 最后看 C 层，决定动作节奏

## 当前版本原则

- A 层信号优先级最高
- C 层信号不能单独改变公司长期结论
- 若 A 层和 B 层恶化，再结合 C 层进入“卖出复核”
