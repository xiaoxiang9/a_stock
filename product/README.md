# 项目产物

这个目录是最终可独立部署运行的投资复盘系统。

## 分层

- `app/`：表达层，负责前端页面、后端接口和消息触达。
- `config/`：配置层，负责模型、数据源、邮件、任务、部署参数。
- `core/`：核心层，负责投研分析 agent、模型调用和通用抽象。
- `data/`：数据层，负责数据获取、校验、数据库操作和关键数据清单。
- `jobs/`：任务层，负责定时刷新、报告发送和调研触发。
- `modules/`：业务层，负责宏观、行业、好公司池、交易记录、公司调研等模块。
- `reports/`：报告产物，负责日报、周报、月报和专题报告。
- `tools/`：工具层，负责通用工具和校验脚本。
- `scripts/`：脚本层，负责安装、初始化、部署和迁移。
- `tests/`：测试。
- `docs/`：产物文档。

新增项目产物文件前，先阅读 `records/project_structure.md`。
