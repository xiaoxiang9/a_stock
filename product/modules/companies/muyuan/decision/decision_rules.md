# 牧原股份五档决策规则

## 五档定义

- 观察：未建仓，判断是否达到好公司标准
- 持有：已判断为好公司，当前无需操作；可为已持仓，也可为空仓等待买点
- 加仓观察：好公司，且价格 / 周期 / 经营 / 热度条件更有利，进入重点准备动作区
- 减仓观察：公司仍然优秀，但短期热度偏高或涨幅过快，考虑部分收益落袋为安
- 卖出复核：核心逻辑可能发生变化，需要重新评估是否继续持有，必要时考虑清仓

## 状态切换规则

### 观察 -> 持有

满足以下条件后进入持有：

- 公司通过好公司评分，综合分不低于 70
- 主要风险可识别、可跟踪
- 没有出现明显治理硬伤

### 持有 -> 加仓观察

满足以下一组或多组信号时进入加仓观察：

- 猪周期位置更有利，行业悲观但经营未恶化
- 公司月度经营数据稳健，成本优势继续兑现
- 估值回落到更有吸引力的历史区间
- 热度不高，交易拥挤度可接受

### 持有 -> 减仓观察

满足以下一组或多组信号时进入减仓观察：

- 短期涨幅过快，热度显著抬升
- 换手率、成交额、情绪同步偏热
- 估值已明显偏离历史舒适区
- 公司质地未变，但赔率下降

### 任一状态 -> 卖出复核

一旦出现以下问题，直接进入卖出复核：

- 公司核心成本优势被破坏
- 月度经营与行业周期明显背离且持续恶化
- 资产负债表快速恶化
- 重大治理、财务、合规问题出现
- 原有投资逻辑不再成立

## 当前适用结论

截至 2026-06-27，牧原股份处于“持有”状态：公司已进入好公司池，但交易层仍需等待更优周期与赔率配合。
