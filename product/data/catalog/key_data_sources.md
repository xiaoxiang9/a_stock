# 关键数据来源入口

本文件保留为数据目录入口，避免历史引用失效。

当前项目的数据层不再维护“某个标的专属数据清单”，而是维护通用数据能力。股票代码、公司名称、行业、时间范围等都应作为查询参数传入。

详细清单见：

- [通用数据能力清单](./data_capabilities.md)

## 数据层原则

- 数据层描述“有哪些数据能力、如何查询、来源和口径是什么”，不绑定具体股票。
- 当前牧原股份日报只是数据能力调用场景之一，后续麦格米特等标的应复用同一套数据能力。
- 数据获取优先级为：固化数据集 / 本地可信快照 > 已代码固化接口或 fetcher > Tushare > 东方财富 skill > 官方/交易所/监管/公司公告 > 权威财经网站 > 互联网搜索。
- 任何进入模型分析的数据，都必须带来源、时间、口径和可信等级。
