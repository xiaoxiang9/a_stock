# 估值 Agent

估值 Agent 负责根据输入的股票代码或公司名称，结合外部证据，输出结构化估值结果。

## 设计边界

- 不直接调用 `product.data` 的取数实现
- 不读取 `app` 子系统的配置文件
- 只接收上游提供的证据包，再做分析和收敛
- 若需要调用 skill 侧的证据脚本，优先使用仓库 `skills/released/` 里的固化副本，再回退到机器本地安装态

## 输入

- 股票代码或公司名称
- 估值分析日期
- 当前市值
- 可选的估值指标和外部证据

## 输出

- 估值状态
- 估值结论
- 估值区间
- 分析依据
- 风险点
- 缺失数据和后续补数问题
