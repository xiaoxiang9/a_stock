# 每晚 01:12 日复盘说明

## 目标

每天凌晨 01:12 固定输出一份当日复盘，服务第二天的投资决策，并持续评估指标是否有效。

## 使用方式

- 模板文件：`product/reports/daily/muyuan_21_template.md`
- 输出文件命名：`product/reports/daily/YYYY-MM-DD-muyuan.md`
- 数据顺序：Tushare 主数据 -> 东方财富校验 -> 官方公告/公开网页补充

## 最低输出要求

1. 必须给出一句话结论
2. 必须给出关键数据快照
3. 必须标注主来源与校验来源
4. 必须明确当前五档状态是否变化
5. 必须写出第二天重点跟踪点

## 模板设计原则

- 先结论，后数据
- 先周期，后估值
- 先事实，后判断
- 保留“指标有效性检查”栏目，便于实战中持续优化框架
