# 项目引用 skill 清单

本文件记录本项目当前引用、依赖或约定优先使用的 skill。

## 1. 记录口径

这里记录的是“项目会使用到的外部或内部 skill 能力”，不代表这些 skill 已经被抽取为本项目自有能力。

分类说明：

- 已使用：当前代码、日报或研究流程已经实际调用。
- 约定优先：项目约定中明确优先使用，但当前不一定每次自动调用。
- 候选沉淀：未来可能抽取为本项目自有 skill，但当前只记录，不抽取。

## 3. 当前数据优先级中的位置

本项目数据获取优先级：

1. 代码接口获取
2. Tushare skill 获取
3. 东方财富 skill 获取
4. 官方、交易所、监管、公司公告等权威途径获取
5. 互联网搜索，多信源验证

其中：

- Tushare 是主数据来源。
- 东方财富妙想系列主要承担二次校验和金融场景搜索。
- 如果代码接口已固化，优先使用代码接口，减少人工 skill 调用不确定性。

## 6. 部署提示

独立部署时需要确认：

- 是否配置 `TUSHARE_TOKEN`
- 是否配置 `MX_APIKEY`
- 是否安装或可访问东方财富妙想系列 skill
- 如果某个 skill 不可用，系统应降级为已固化代码接口、官方来源或多信源人工校验
